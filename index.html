Feature Card Demo

This entire card is styled with ChaiCSS utility classes! Each element uses chai-* classes that are parsed at runtime and converted to inline styles. The classes are then removed automatically, leaving a clean DOM structure with applied styles.

✓ Smart value parsing: numbers get px, units are preserved
✓ Multi-part values: chai-border-2px-solid-blue works perfectly
✓ Shorthand properties: m, p, mt, mb, ml, mr, pt, pb, pl, pr all supported
